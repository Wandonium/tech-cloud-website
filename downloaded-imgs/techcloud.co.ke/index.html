iSecurity is a platform to scan and manage all visitors using their national ID when entering and leaving private and public building. Currently this process is done manually requiring the security officers to write the ID details on a book which is often discarded once it is full. This platform would allow all building fitted with our system to be interconnected via the Cloud and linked to a national database, which would correctly identify each visitor entering and leaving any connected building as well as detecting fraudulent IDs and personal documentation.

A smart farming Internet of Things solution that targets greenhouse farmers. It employs precision farming method by capturing environmental factors that affect the wellbeing of the crops grown inside a green house. After capturing the sensor data, insights are drawn from it by analyzing it using a machine learning model and previous historical data. From the insights the solution is able to advice the farmer accordingly as well as give recommendations. Other than the recommendations the farmer gets, he/she is able to monitor a selected few parameters from his/her farm and get SMS alerts whenever those parameters change. Some of these environmental factors include humidity, temperature, atmospheric pressure , soil moisture and soil Fertility
